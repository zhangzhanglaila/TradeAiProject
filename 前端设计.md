# 前端设计

项目背景
开发一个基于 Vue 3 + Element Plus 的管理后台，用于展示和管理哈萨克斯坦贸易数据与新闻数据。系统需适配后端 API 接口，实现用户登录、数据看板、数据管理、AI 问答等核心功能。

技术栈要求
框架：Vue 3 + Vite

UI库：Element Plus

状态管理：Pinia

路由：Vue Router 4

HTTP请求：Axios

图表：ECharts

工具库：Day.js（日期处理）、lodash

项目结构建议
text

页面功能详细设计
1. 登录页 /login
功能点：

用户名、密码输入框

验证码输入框 + 图片验证码（点击刷新）

登录按钮（表单验证）

记住密码选项（可选）

交互说明：

进入页面自动获取验证码

登录成功后跳转首页看板

错误提示使用 Element Plus 的 Message 组件

2. 首页看板 /dashboard（所有登录用户）
布局：上中下结构

顶部：欢迎语、当前时间、退出登录

中部卡片区（3-4个）：

总记录数卡片

总贸易金额卡片（格式化显示，如 ¥1,234,567）

贸易方式数量卡片

图表区（两列布局）：

左侧：折线图 - 月度贸易趋势（最近12个月）

右侧：饼图 - 贸易方式金额占比

底部：最新数据表格（最近5条贸易记录）

接口调用：

总览统计：GET /api/stats/overview

月度趋势：GET /api/stats/trend?startMonth&endMonth（默认最近12个月）

贸易方式占比：GET /api/stats/trade-mode-ratio

最新数据：GET /api/trade/page?current=1&size=5

3. 贸易数据管理 /trade（管理员可见）
3.1 数据导入
文件上传组件（拖拽或点击）：

支持 CSV 格式

上传时需选择：国家、进出口方向、合并模式（跳过/覆盖）

可选设置最大导入行数（默认200）

上传进度条显示

3.2 数据查询
搜索表单（一行布局）：

年月范围选择器（YYYYMM格式）

国家下拉框

进出口方向下拉框（全部/进口/出口）

关键词搜索（模糊匹配商品/伙伴名称）

搜索按钮 + 重置按钮

导出按钮（权限控制：管理员可见）

3.3 数据表格
列字段：年月、国家、贸易流向、商品名称、贸易方式、人民币金额、操作

支持分页（后端分页）

操作列：详情、编辑、删除（权限控制）

批量删除：支持勾选框

3.4 新增/编辑弹窗
表单字段对应 trade_data 表主要字段

表单验证（必填、金额格式等）

4. 新闻数据管理 /news（管理员可见）
类似贸易数据管理，但字段不同：

表格列：标题、发布日期、标签、操作

详情弹窗显示完整内容（HTML预览）

搜索条件：标题关键词、日期范围

5. 统计分析 /stats（所有用户）
功能点：

顶部筛选条件：

国家多选（或全选）

进出口方向

年月范围

图表区域（Tab切换）：

Tab1 - 趋势图：折线图展示金额变化

Tab2 - 贸易方式分布：饼图/柱状图

Tab3 - 国家分布：横向柱状图（Top10）

6. AI智能问答 /ai（所有用户）
布局：类似聊天界面

左侧：历史对话列表（分页加载）

右侧：当前对话区域

消息列表（用户问题左对齐，AI回答右对齐）

底部输入框 + 发送按钮

加载状态显示

功能点：

发送问题调用 /api/ai/ask 接口

实时显示回答（目前为占位回复 + 数据摘要）

历史记录分页加载（滚动加载）

清空对话按钮

状态管理（Pinia）
1. 用户状态 userStore
javascript
state: {
    token: '',
    userInfo: { username, role },
    permissions: []
}
actions: {
    login(),
    logout(),
    getCaptcha()
}
2. 应用状态 appStore
javascript
state: {
    collapsed: false,  // 侧边栏折叠
    breadcrumbs: [],
    tagsView: []       // 多标签页（可选）
}
路由权限控制
白名单：/login

动态路由：根据角色（ADMIN/USER）动态添加

ADMIN：所有路由

USER：/dashboard、/stats、/ai

路由守卫：检查 token 和角色权限

Axios 封装要求
基础URL配置（环境变量）

请求拦截器：自动添加 token

响应拦截器：

统一错误处理（401跳转登录，其他错误提示）

加载状态管理（可选）

下载文件处理（导出接口）

通用组件封装
1. CaptchaImage.vue
接收 uuid 和 base64 图片

点击刷新验证码

2. UploadCsv.vue
封装文件上传逻辑

进度条显示

错误处理

3. DataTable.vue
封装分页表格

支持自定义列

操作按钮插槽

4. DateRangePicker.vue
封装年月范围选择

返回 YYYYMM 格式数组

UI/UX 要求
主题：默认 Element Plus 主题，可自定义主色（建议商务蓝）

响应式：适配 1366x768 以上分辨率

加载状态：表格、按钮、页面切换需有 loading 效果

空状态：表格无数据时显示占位图

错误处理：友好错误提示，网络异常重试按钮

表单验证：实时反馈错误信息

接口联调注意事项
所有请求需携带 token（Authorization: Bearer）

日期格式统一为 YYYYMM（如 202501）

分页参数：current, size

文件上传：Content-Type: multipart/form-data

导出接口：处理文件流下载

开发环境要求
Node.js 16+

VSCode + Volar 插件

提供 .env.development 配置 API 地址
